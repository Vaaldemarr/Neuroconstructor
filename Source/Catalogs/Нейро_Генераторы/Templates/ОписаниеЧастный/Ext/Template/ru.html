Принцип работы частного генератора такой-же как и у общего генератора за исключением того, что он специализирован под один слой.
Частный генератор может вызываться из общего генератора при помощи команды next. Например:
next_batch1=next(train_generator1)

Так вызывается генератор данных изображений:
next_batch1=train_generator1.next()

Встроенный генератор временных рядов вызывается так:
next_batch1=train_generator1[i]
где i - номер шага за эпоху, а всего шагов len(train_generator1)-1

Параметры:

samples

-

массив с данными, указанный для входного слоя

targets

-

массив с данными, указанный для выходного слоя

min_index

-

индекс первого элемента в массиве данных

max_index

-

индекс последнего элемента в массиве данных

shuffle

-

перемешивать данные, принимает значение True или False

batch_size

-

размер пакета. Обычно длина всех данных равна размеру пакета * шагов за эпоху

subset

-

какой в данный момент вызван генератор: значение 'training' - тренировочный и 'validation' - проверочный

Имя переменной для частного генератора назначается автоматически:

·         Для обращения к тренировочному генератору используйте литерал train_generator и номер входного слоя (например, train_generator1)

·         Для обращения к проверочному генератору используйте литерал validation_generator и номер входного слоя (например, validation_generator1)
